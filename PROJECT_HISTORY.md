## 2. CSS дизайн-система

### 2.1 `css/variables.css`
CSS-переменные: тёмная тема с фиолетово-синими градиентами, типографика (Google Fonts — Inter), отступы, размеры, border-radius, тени.

### 2.2 `css/reset.css`
Современный CSS-сброс (box-sizing, margin, padding, шрифт по умолчанию).

### 2.3 `css/base.css`
Базовая типографика, стили ссылок, прокрутка (smooth scroll).

### 2.4 `css/layout.css`
Контейнеры (`.container`), сетки (CSS Grid/Flexbox), секционная структура.

### 2.5 `css/components.css`
Кнопки с градиентными hover-эффектами, карточки фич с glassmorphism, навигация (header/footer), badge-компоненты.

### 2.6 `css/animations.css`
Fade-in при скролле, hover-эффекты, floating-анимации для hero-секции, плавное появление элементов.

### 2.7 `css/home.css`
Стили главной страницы: hero-градиент, фичи-сетка, download-секция.

### 2.8 `css/legal.css`
Стили юридических страниц: удобная типографика, содержание, навигация.

---

## 3. HTML-страницы

### 3.1 `index.html` — Главная страница
6 секций:
1. **Header** — фиксированный, прозрачный фон с blur при скролле, лого + навигация + переключатель языков
2. **Hero** — мокап телефона с имитацией чата, градиентный заголовок, CTA-кнопки
3. **Features** — 6 карточек с иконками (шифрование, звонки, каналы, группы, стикеры, файлы) с анимацией появления при скролле
4. **Security** — акцент на безопасности, end-to-end шифрование (3 элемента)
5. **Download** — кнопки App Store / Google Play (бейджи «Скоро») + статистика
6. **Footer** — ссылки, копирайт, социальные сети

### 3.2 `privacy.html` — Политика конфиденциальности
Юридическая страница с навигацией по разделам. Переведена на 3 языка.

### 3.3 `terms.html` — Пользовательское соглашение
Юридическая страница с условиями использования. Переведена на 3 языка.

---

## 4. JavaScript модули

### 4.1 `js/main.js`
Точка входа: импорт и инициализация всех модулей.

### 4.2 `js/components/header.js`
Мобильное hamburger-меню, эффект фиксированного header при скролле, активная ссылка.

### 4.3 `js/components/animations.js`
Scroll-анимации через `IntersectionObserver` — плавное появление секций при прокрутке.

### 4.4 `js/components/i18n.js`
Переключатель языков (RU/UZ/EN). Загрузка JSON-файлов переводов, сохранение выбора в localStorage.

### 4.5 `js/utils/helpers.js`
Утилиты: throttle, debounce, smooth scroll.
